IT Home Lab Project

A hands on home lab built to develop practical skills for IT support and platform engineering roles. The project covers on-premises Windows Server administration, Active Directory, Group Policy, ServiceNow ITSM and Microsoft 365 cloud administration.

All tasks are fully documented with screenshots and step by step instructions in the project log attached to this repository.
